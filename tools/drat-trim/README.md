# Vendored DRAT checker

This directory contains `drat-trim.c` from Marijn Heule and Nathan Wetzler's
[`drat-trim`](https://github.com/marijnheule/drat-trim) repository, pinned at
commit:

```text
2e3b2dc0ecf938addbd779d42877b6ed69d9a985
```

The source SHA-256 is:

```text
d834b649f437e091597f5347f259b9f681087f89ca0844d0cee250a1a1a0c2ee
```

It is vendored so that the order-98 nonexistence certificates can be checked
from a clean checkout without downloading or trusting a prebuilt executable.
The upstream MIT license is preserved in `LICENSE`.
